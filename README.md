# Shubh.Online

Personal pharma and chemistry knowledge website for API research, medicinal chemistry, software tools, interview preparation, and study libraries.

## Sections

- Home profile with typewriter introduction
- My Library with integrated Chemistry Library content
- Organic Chemistry atlas and functional group conversion notes
- Medicinal Chemistry folder-style subject library
- Software toolkit for computational chemistry and scientific modeling
- Department-wise pharma interview preparation
- Bio / portfolio and contact page

## Run Locally

Open `index.html` in a browser, or serve the folder with any static server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## GitHub Pages

Upload this folder as the repository root. The site uses relative paths, so GitHub Pages can serve it directly from `index.html`.

## Notes

The `Chemistry Library` folder is included inside this website folder so all required files stay together for GitHub upload.
